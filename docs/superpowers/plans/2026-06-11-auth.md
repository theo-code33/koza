# Auth.js login/signup (sous-projet #3) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans. Steps use checkbox (`- [ ]`).

**Goal:** Vraie authentification (Auth.js v5, Credentials email+mot de passe, JWT, bcrypt) : pages `/login` et `/signup`, middleware de protection, déconnexion, et remplacement du stub `getCurrentUserId()` par `auth()`. Débloque le signup en direct sur le lien déployé.

**Architecture:** `src/auth.ts` expose `{ auth, handlers, signIn, signOut }`. Le provider Credentials vérifie email + `bcrypt.compare`. JWT porte `user.id`. `middleware.ts` protège tout sauf les routes publiques. `getCurrentUserId()` lit `auth()`.

**Tech Stack:** next-auth 5 (beta), bcryptjs (déjà installé), Zod, RHF, next-intl.

- [ ] **Step 3 : `src/auth.ts`**
```ts
import NextAuth from "next-auth";
import Credentials from "next-auth/providers/credentials";
import bcrypt from "bcryptjs";
import { prisma } from "@/lib/prisma";

export const { handlers, auth, signIn, signOut } = NextAuth({
  session: { strategy: "jwt" },
  pages: { signIn: "/login" },
  providers: [
    Credentials({
      credentials: { email: {}, password: {} },
      authorize: async (credentials) => {
        const email = typeof credentials?.email === "string" ? credentials.email : "";
        const password = typeof credentials?.password === "string" ? credentials.password : "";
        if (!email || !password) return null;
        const user = await prisma.user.findUnique({ where: { email } });
        if (!user || !user.passwordHash) return null;
        const valid = await bcrypt.compare(password, user.passwordHash);
        if (!valid) return null;
        return { id: user.id, email: user.email };
      },
    }),
  ],
  callbacks: {
    jwt({ token, user }) {
      if (user?.id) token.id = user.id;
      return token;
    },
    session({ session, token }) {
      if (token.id && session.user) session.user.id = token.id as string;
      return session;
    },
  },
});
```

```ts
"use server";

import bcrypt from "bcryptjs";
import { prisma } from "@/lib/prisma";
import { signIn } from "@/auth";
import { credentialsSchema } from "@/lib/validators";

export async function signupAction(email: string, password: string): Promise<{ error?: string }> {
  const parsed = credentialsSchema.safeParse({ email, password });
  if (!parsed.success) return { error: "invalid" };
  const existing = await prisma.user.findUnique({ where: { email } });
  if (existing) return { error: "emailTaken" };
  const passwordHash = await bcrypt.hash(password, 10);
  await prisma.user.create({ data: { email, passwordHash, settings: { create: {} } } });
  await signIn("credentials", { email, password, redirectTo: "/dashboard" });
  return {};
}

export async function loginAction(email: string, password: string): Promise<{ error?: string }> {
  try {
    await signIn("credentials", { email, password, redirectTo: "/dashboard" });
    return {};
  } catch (error) {
    // signIn relance une redirection en cas de succès ; sinon credentials invalides.
    if (error instanceof Error && error.message.includes("NEXT_REDIRECT")) throw error;
    return { error: "invalidCredentials" };
  }
}
```
> Note : `signIn` lève un `NEXT_REDIRECT` en cas de succès (à relancer) ; une `CredentialsSignin` en cas d'échec → message d'erreur.

## Phase 3 — Middleware & déconnexion

### Task 3 : protection des routes + bouton déconnexion

**Files:** Create `middleware.ts` ; Modify `src/app/(main)/settings/page.tsx`, `src/components/settings/*`, `src/locales/{fr,en}.json`

- [ ] **Step 1 : `middleware.ts`** (racine) :
```ts
import { auth } from "@/auth";

const PUBLIC = new Set(["/", "/login", "/signup", "/privacy", "/terms"]);

export default auth((req) => {
  const { nextUrl } = req;
  const isPublic =
    PUBLIC.has(nextUrl.pathname) ||
    nextUrl.pathname.startsWith("/api/auth") ||
    nextUrl.pathname.startsWith("/api/health");
  if (isPublic || req.auth) return;
  if (nextUrl.pathname.startsWith("/api")) {
    return new Response("Unauthorized", { status: 401 });
  }
  return Response.redirect(new URL("/login", nextUrl));
});

export const config = {
  matcher: ["/((?!_next/static|_next/image|favicon.ico|.*\\.[^/]+$).*)"],
};
```
